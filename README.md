# MintTea-neofetch

Custom neofetch for Min Tea OS

## How to

###### Download The Package
* Download the script
* Open Terminal
* $ chmod +x install 
* $ sudo ./install

###### Clone
* Open terminal
* git clone https://github.com/Litle-Panda/MintTea-neofetch.git
* $ cd MintTea-neofetch
* $ chmod +x install
* $ sudo ./install
